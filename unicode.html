Encoding

To make it easier to develop fonts, the tools/zbalermorna.ps encoding can be loaded into FontForge to see all of the characters used in Zbalermrona texts. The optional Latin punctuation is listed first. The order/arrangement here only matters if you use a 16-character-wide display in FontForge. This is really only useful after the names/code points are created - a separate process is needed for that currently.
